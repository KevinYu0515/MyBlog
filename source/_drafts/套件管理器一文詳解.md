## 一些實用指令

- 將 node_modules 移除
```powershell
cmd /c "rmdir /s /q node_modules"
```

```powershell
mkdir empty
robocopy empty node_modules /MIR
rmdir empty
rmdir node_modules
```

- 將多個專案內的 node_modules 刪除
```powershell
Get-ChildItem -Recurse -Directory -Filter node_modules |
ForEach-Object {
  cmd /c "rmdir /s /q `"$($_.FullName)`""
}
```
